About Clockforce

Meeting time finder, timezone comparison & MCE date conversion tool

Find the best meeting times across timezones with the interactive timeline view. Drag to select time ranges, save Time Blocks and click them to preview across all timezones. Compare sunrise/sunset across locations. Switch between Timeline and Clocks views with 12h/24h/Mix time format support.

Use the floating datetime picker to lock all clocks to any point in time in any timezone. Generate production-ready date conversion scripts for Salesforce Marketing Cloud Engagement in SQL, AMPScript, and SSJS with DST-aware logic.

Save and load configurations, share via URL, and export/import as JSON. Built with vanilla JavaScript and the Salesforce Lightning Design System.
